
# Appcode安装要求

- <a href="#系统要求">系统要求</a>
- <a href="#安装">安装</a>
<h2 id="硬件要求">硬件要求</h1>
- 2GB内存以上
<h2 id="系统要求">系统要求</h2>
- OS X 10.10或者之后
- Xcode 7.2或者之后
<h2 id="安装">安装</h2>
1.从[AppCode官网](https://www.jetbrains.com/objc/download/)下载`AppCode-*.dmg` OSX安装的源文件
</br>
2.双击下载`AppCode-*.dmg` OSX安装的源文件
</br>
3.拷贝**Appcode**到`应用程序`

>Appcode只支持JDK1.8版本
